🎵 SoundHub
Seu centro musical — todos os apps de música em um único lugar.
App Web instalável (PWA) com player, atalhos para Spotify, YouTube Music, Deezer, Amazon Music e muito mais.
✨ Funcionalidades
🎧 Atalhos rápidos para todos os seus apps de música
▶️ Player de demonstração com controles de play/pause, próxima e anterior
➕ Adicionar/remover apps com um toque — preferências salvas localmente
📋 Playlists recentes organizadas visualmente
📲 Instalável como app no celular (PWA) via Chrome/Safari
🌐 Funciona offline graças ao Service Worker
🚀 Como publicar no GitHub Pages
1. Criar o repositório
Bash
2. Ativar GitHub Pages
Vá em Settings → Pages
Em "Source", selecione Deploy from a branch
Escolha a branch main e a pasta / (root)
Clique em Save
Após alguns minutos, seu app estará em:
Código
3. Instalar no celular
Android (Chrome):
Abra o link acima no Chrome
Toque no menu (⋮) → "Adicionar à tela inicial"
Confirme — o ícone do SoundHub aparecerá na tela inicial
iPhone (Safari):
Abra o link no Safari
Toque em Compartilhar (ícone de caixa com seta)
"Adicionar à Tela Inicial" → Adicionar
